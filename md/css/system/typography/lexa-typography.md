## 🅰️ Lexa — Tipografia do Sistema

**Nível / Level:** Adulto / Adult

- **PT:** Lexa define a base tipográfica do sistema.
  Ela estabelece fontes, hierarquia textual, pesos e
  _line-height_, garantindo consistência, legibilidade
  e uma linguagem visual unificada em todas as seções
  e componentes.

- **EN:** Lexa defines the system typography foundation.
  She establishes fonts, text hierarchy, weights and
  _line-height_, ensuring consistency, readability
  and a unified visual language across all sections
  and components.
