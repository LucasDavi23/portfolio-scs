## ⭐ Vega — UI do Sistema

**Nível / Level:** Adulto / Adult

- **PT:** Vega define os componentes visuais base do sistema.
  Ela estabelece padrões de UI compartilhados, incluindo
  botões, links, estados de interação (hover, focus, disabled)
  e microcomponentes reutilizáveis utilizados em todas
  as seções da interface.

- **EN:** Vega defines the system base UI components.
  She establishes shared UI patterns, including
  buttons, links, interaction states (hover, focus, disabled)
  and reusable microcomponents used across the interface.
