# 🌫️ Veil — Message Box Skin

**Nível / Level:** Jovem / Young

**PT:**  
Veil define a aparência visual da **Message Box** do sistema.

Ela controla cores, transições e animações da interface,
garantindo que as mensagens exibidas pelo **Halo** tenham
uma apresentação suave e consistente.

Veil atua apenas na camada visual e não contém lógica
de comportamento.

**EN:**  
Veil defines the visual appearance of the system **Message Box**.

It controls colors, transitions and animations,
ensuring messages displayed by **Halo** have a smooth
and consistent presentation.

Veil operates only in the visual layer and contains
no behavioral logic.
