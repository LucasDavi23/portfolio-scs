## 🧱 Foundation — Regras Estruturais Globais CSS

**Tipo / Type:** Estrutural / Structural

**PT:**  
Foundation define as regras estruturais globais do sistema.

Este arquivo estabelece fundamentos utilizados em todo o layout,
incluindo:

- controle de viewport
- comportamento base de scroll
- estilos estruturais globais
- mecanismos como travamento de scroll em contextos de modal

Foundation sustenta todas as demais camadas visuais do sistema.

**EN:**  
Foundation defines the global structural rules of the system.

This file establishes base behaviors used across the layout,
including:

- viewport handling
- base scroll behavior
- global structural styles
- mechanisms such as scroll locking in modal contexts

Foundation supports all other visual layers of the system.
