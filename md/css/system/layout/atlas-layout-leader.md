## 🧭 Atlas — Líder de Layout do Sistema

**Nível / Level:** Adulto / Adult

- **PT:** Atlas coordena a estrutura de layout do sistema. Ela não estiliza
  componentes ou features específicas, mas define containers, grids
  e limites estruturais que sustentam todas as páginas e módulos.

- **EN:** Atlas coordinates the system layout structure. She does not style
  individual components or specific features, but defines containers,
  grids and structural boundaries that support all pages and modules.
