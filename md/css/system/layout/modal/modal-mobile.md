## 📱 Modal — Ajustes Mobile (CSS)

**Tipo / Type:** Arquivo Estrutural / Structural File

**PT:**  
Arquivo técnico responsável por ajustes específicos de
layout e comportamento de modais em dispositivos móveis.

Ele aplica refinamentos de:

- espaçamento
- altura
- controle de scroll
- adaptação visual para telas menores

Garantindo melhor usabilidade em dispositivos mobile.

**EN:**  
Technical stylesheet responsible for mobile-specific
modal adjustments.

It handles refinements for:

- spacing
- height
- scroll control
- visual adaptation for smaller screens
