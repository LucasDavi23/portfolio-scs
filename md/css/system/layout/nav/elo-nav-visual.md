# 🎨 Elo — Navigation Visual Connector

**Nível / Level:** Adulta / Adult

**PT:**  
Elo é responsável por representar visualmente o estado ativo da navegação.

Ela conecta o estado lógico definido pelo JavaScript com a interface,
aplicando os estilos que indicam qual seção ou link está ativo no menu.

Elo atua exclusivamente na camada visual, refletindo o estado da navegação
sem interferir na lógica de comportamento.

**EN:**  
Elo is responsible for visually representing the active navigation state.

She bridges the logical state defined by JavaScript with the interface,
applying the styles that indicate which section or link is currently active.

Elo operates exclusively in the visual layer, reflecting the navigation
state without controlling navigation logic.
