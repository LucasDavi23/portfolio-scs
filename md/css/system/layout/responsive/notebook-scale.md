## 💻 Notebook Scale — Ajustes de Escala para Notebooks

**Tipo / Type:** Estrutural / Structural

**PT:** Arquivo técnico responsável por ajustes de escala visual
para resoluções comuns de notebooks.

Ele aplica estratégias de:

- **zoom** como abordagem principal
- **scale** como fallback

Essa camada evita o uso de larguras fixas e preserva
as proporções e a legibilidade do layout em telas
entre **1280–1600px**.

**EN:** Technical stylesheet responsible for visual scaling
adjustments for common notebook resolutions.

It applies:

- **zoom** as the primary strategy
- **scale** as a fallback

This layer avoids fixed widths and preserves
layout proportions and readability on screens
between **1280–1600px**.
