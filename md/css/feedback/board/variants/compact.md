## 🧩 Compact — Board Layout Variant CSS

**Tipo / Type:** Ferramenta / Tool

**Liderança:** Seren (Board Leader)

**PT:** Variante compacta do board de feedback.  
O modo **compact** reduz espaçamentos, margens e densidade dos cards,
permitindo visualizar mais avaliações na tela e otimizar o uso
do espaço vertical em listas longas.

**EN:** Compact layout variant for the feedback board.  
The **compact** mode reduces spacing, margins and card density,
allowing more reviews to be displayed on screen and improving
vertical space usage for long lists.
