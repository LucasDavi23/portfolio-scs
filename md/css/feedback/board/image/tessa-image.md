## 🖼️ Tessa — Imagem do Feedback CSS

**Nível / Level:** Jovem / Young

- **PT:** Tessa é responsável pela apresentação visual da imagem dentro
  do card de feedback. Ela define o enquadramento do thumbnail,
  proporção (1:1), comportamento de crop, uso de object-fit,
  fallback visual e bordas da imagem, garantindo consistência
  e boa leitura visual.

- **EN:** Tessa is responsible for the visual presentation of images
  inside the feedback card. She defines thumbnail framing,
  aspect ratio (1:1), crop behavior, object-fit usage,
  visual fallback and image borders, ensuring consistency
  and visual clarity.
