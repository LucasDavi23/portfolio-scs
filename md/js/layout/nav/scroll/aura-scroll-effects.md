### 🌬️ Aura — Scroll Effects Module

**Nível / Level:** Aprendiz / Apprentice

**PT:**  
Aura é o módulo reservado para **efeitos visuais relacionados ao scroll da página** dentro do layout.

Atualmente não possui lógica ativa e funciona como um **ponto de expansão arquitetural** para comportamentos baseados em rolagem.

Possíveis responsabilidades futuras:

- aplicar **mudanças visuais no header**
- ativar **sombras dinâmicas**
- alternar **classes condicionais baseadas em scroll**
- centralizar qualquer comportamento dependente da **rolagem da página**

Aura existe para evitar que lógica de scroll fique **espalhada pelo sistema**.

---

**EN:**  
Aura is the module reserved for **scroll-based visual effects** within the layout.

Currently it has no active logic and serves as an **architectural expansion point** for behaviors triggered by page scrolling.

Possible future responsibilities:

- apply **header visual changes**
- enable **dynamic shadows**
- toggle **scroll-based classes**
- centralize any **page scroll related behavior**

Aura exists to prevent scroll logic from being **scattered across the system**.
