# 🧭 Nara — Navigation State Specialist

**Nível / Level:** Adulto / Adult

**PT:**  
Nara detecta a **seção ativa da página** através do scroll,
utilizando `IntersectionObserver` para manter o estado da navegação sincronizado.

Responsabilidades:

- identificar qual **seção está visível na tela**
- atualizar o **estado ativo da navegação**
- aplicar a **classe correspondente nos links do menu**

Nara cuida apenas do **estado lógico da navegação**.

A representação visual do estado ativo é responsabilidade da **Elo (CSS)**.

---

**EN:**  
Nara detects the **active section of the page** through scroll,
using `IntersectionObserver` to keep the navigation state synchronized.

Responsibilities:

- detect which **section is currently visible**
- update the **navigation active state**
- apply the **corresponding class to navigation links**

Nara handles only the **navigation state logic**.

The visual representation of the active state is handled by **Elo (CSS)**.
