# 🌑 Nyx — Form Modal Controller (Layout)

**Nível / Level:** Adulto / Adult

---

## 📖 Visão Geral / Overview

**PT:**  
Nyx controla a **abertura e o fechamento dos modais de formulário** dentro do layout.

**EN:**  
Nyx controls the **opening and closing of form modals** within the layout.

---

## 🧠 Responsabilidades / Responsibilities

**PT:**

- gerenciar o **overlay do modal**
- definir o **foco inicial ao abrir**
- permitir fechamento via **tecla ESC**
- aplicar **bloqueio de scroll** enquanto o modal estiver ativo

**EN:**

- manage the **modal overlay**
- define the **initial focus on open**
- allow closing via **ESC key**
- apply **scroll locking** while the modal is active

---

## 🚫 Limites de Atuação / Scope Boundaries

**PT:**

Nyx atua apenas no **controle estrutural do modal**.

Ela não:

- valida dados
- controla envio de formulário
- manipula regras de domínio

**EN:**

Nyx operates only in the **structural control of the modal**.

She does not:

- perform data validation
- handle form submission logic
- manipulate domain rules

---

## 🔗 Integração no Sistema / System Integration

Nyx atua no **Layout Layer**, servindo como ponte entre:

- interação do usuário (UI)
- comportamento visual do modal

---

## 🧩 Observações

**PT:**  
Nyx substitui a antiga Lyra neste contexto, mantendo a mesma função com nomenclatura mais alinhada à arquitetura atual.

**EN:**  
Nyx replaces the former Lyra in this context, maintaining the same role with naming aligned to the current architecture.
