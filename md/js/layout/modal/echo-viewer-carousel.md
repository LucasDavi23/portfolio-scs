# 🔊 Echo — Viewer Carousel Assistant

**Nível / Level:** Aprendiz / Apprentice

**PT:**  
Echo auxilia a **Iris** quando o modal de imagem opera como **galeria**.

Responsabilidades:

- gerenciar a **lista de imagens**
- controlar o **índice atual**
- permitir navegação entre imagens (**next/prev, swipe e teclado**)
- notificar a **Iris** sempre que a imagem exibida mudar

Echo não abre ou fecha o modal,  
não controla scroll e  
não altera a interface.

---

**EN:**  
Echo assists **Iris** when the image modal operates as a **gallery**.

Responsibilities:

- manage the **image list**
- control the **current index**
- allow navigation between images (**next/prev, swipe and keyboard**)
- notify **Iris** whenever the displayed image changes

Echo does not open or close the modal,  
does not handle scroll and  
does not alter the interface.
