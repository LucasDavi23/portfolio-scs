# 🧩 UUID — Unique Identifier Utility

**Tipo / Type:** Ferramenta / Tool

**PT:**  
UUID é um utilitário do sistema responsável por **gerar identificadores únicos**.

Responsabilidades:

- gerar **IDs únicos** para componentes ou instâncias
- fornecer identificadores para **operações internas**
- utilizar **mecanismos de fallback** quando necessário

UUID atua apenas como **utilitário de geração de identificadores**.

Ele não possui estado interno  
e não contém lógica de negócio.

---

**EN:**  
UUID is a system utility responsible for **generating unique identifiers**.

Responsibilities:

- generate **unique IDs** for components or instances
- provide identifiers for **internal operations**
- use **fallback generation mechanisms** when needed

UUID operates only as an **identifier generation utility**.

It has no internal state  
and contains no business logic.
