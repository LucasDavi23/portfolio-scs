# 🌟 Halo — Message Box UI Specialist

**Nível / Level:** Jovem / Young

**PT:**  
Halo é o **componente global de mensagens flutuantes do sistema**.

Responsabilidades:

- criar e remover a **message box**
- controlar o **tempo de exibição das mensagens**
- aplicar **animações de entrada e saída**
- manter o **posicionamento alinhado ao `<main>`**

Halo atua apenas como **camada visual de comunicação com o usuário**.

Ele não executa lógica de negócio,  
não dispara ações do sistema  
e não controla fluxo de aplicação.

---

**EN:**  
Halo is the **system’s global floating message box component**.

Responsibilities:

- create and remove the **message box**
- control the **message display duration**
- apply **entry and exit animations**
- keep the **position aligned relative to `<main>`**

Halo operates only as a **visual communication layer for the user**.

He does not execute business logic,  
does not trigger system actions  
and does not control application flow.
