# ⭐ Stella — Submit Overlay UI Specialist

**Nível / Level:** Jovem / Young

**PT:**  
Stella fornece um **overlay reutilizável de processamento de envio**.

Responsabilidades:

- bloquear a **interação da interface** durante operações de submit
- comunicar o **estado de processamento ao usuário**
- utilizar o **spinner fornecido pela Luma**
- indicar claramente que uma ação está **em andamento**

Stella atua apenas como **camada visual de estado de processamento**.

Ela não executa envio de dados,  
não chama APIs  
e não participa da lógica de negócio.

---

**EN:**  
Stella provides a reusable **processing overlay for submit actions**.

Responsibilities:

- block **interface interaction** during submit operations
- communicate the **processing state to the user**
- use the **spinner provided by Luma**
- clearly indicate that an action is **in progress**

Stella operates only as a **visual processing state layer**.

She does not send data,  
does not call APIs  
and does not participate in business logic.
