# 🍃 Luma — Loading UI Specialist

**Nível / Level:** Adulto / Adult

**PT:**  
Luma é a **interface global de loading do sistema**.

Responsabilidades:

- fornecer um **padrão reutilizável de estados de carregamento**
- exibir **spinner com label**
- controlar **estado de loading em botões**
- garantir que o navegador **renderize o estado visual antes de operações pesadas**

Luma atua apenas na **camada visual de loading**.

Ela não executa requisições de rede  
e não participa da lógica de dados.

---

**EN:**  
Luma is the **system’s global loading UI specialist**.

Responsibilities:

- provide a **reusable loading state pattern**
- display **spinner with label**
- control **button loading states**
- ensure the browser **paints the loading state before heavy operations**

Luma operates only in the **visual loading layer**.

She does not perform network requests  
and does not participate in data logic.
