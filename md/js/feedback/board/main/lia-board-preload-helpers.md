## 🌱 Lia — Preload do Board (Helper)

**Nível / Level: Aprendiz / Apprentice**

**PT:** Aprendiz do Board responsável por **pré-aquecer a API de feedback**.

Ela executa chamadas leves para manter o servidor ativo, reduzindo o tempo da primeira carga.  
Os endpoints aquecidos são então utilizados por **Selah durante o init do Board**.

**EN:** Board apprentice responsible for **pre-warming the feedback API**.

She performs light requests to keep the backend active, improving the initial load time.  
The warmed endpoints are then used by **Selah during the Board initialization**.
