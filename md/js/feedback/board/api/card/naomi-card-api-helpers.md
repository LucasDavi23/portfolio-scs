### 🌸 Naomi — FeedbackCardAPI

**Nível / Level: Jovem / Young**

**PT:**  
Naomi transforma dados do **Apps Script** em Cards de feedback padronizados para o Board.

Ela fornece:

- **list()** — lista de feedbacks
- **listMeta()** — lista com paginação e total
- **latest()** — últimos feedbacks

Naomi realiza a **normalização completa dos itens**, preparando os dados para que o sistema possa renderizar os Cards corretamente.

Ela trabalha integrada ao **Core da Nádia**, utilizando os dados já obtidos pela camada de rede.

Naomi não executa chamadas de rede pesadas — ela atua apenas como **camada de adaptação de dados**.

**EN:**  
Naomi transforms **Apps Script data** into standardized feedback cards for the Board.

It provides:

- **list()** — list of feedbacks
- **listMeta()** — list with pagination and total count
- **latest()** — latest feedbacks

Naomi performs **full normalization of items**, preparing the data so the system can render the feedback cards properly.

She works integrated with **Nadia's Core**, using the data already fetched by the network layer.

Naomi does not perform heavy network requests — it acts purely as a **data adaptation layer**.
