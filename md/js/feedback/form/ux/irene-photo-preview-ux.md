### ✨ Irene — Photo Preview UX Specialist

**Nível / Level:** Aprendiz / Apprentice

**PT:**  
Especialista na **experiência de preview de foto no formulário**.

Responsabilidades:

- estruturar o preview como um **card visual**
- ajustar automaticamente a `<img>` inserida usando **MutationObserver**
- controlar o **overlay** quando uma imagem está presente
- permitir abrir a imagem em **modal** (ou fallback `<dialog>`)

Irene atua apenas na **camada de experiência visual do preview**.

Não valida inputs (**Sofia** faz isso),  
não processa imagens (**Athena**),  
não controla o input ou label do arquivo (**Daphne**)  
e não realiza chamadas de API ou fluxo.

**EN:**  
Specialist in the **photo preview experience in the form**.

Responsibilities:

- structure the preview as a **visual card**
- automatically adjust the inserted `<img>` using **MutationObserver**
- control the **overlay** when an image is present
- allow opening the image in a **modal** (or `<dialog>` fallback)

Irene operates only in the **preview UX layer**.

She does not validate inputs (**Sofia**),  
does not process images (**Athena**),  
does not control the file input or label UI (**Daphne**)  
and does not call APIs or control the form flow.
