### 👨 Gael — Feedback Sender

**Nível / Level:** Jovem / Young

**PT:**  
Especialista no envio do feedback para o **Apps Script**.

Responsabilidades:

- chamar a API através de **Elis**
- interpretar as respostas técnicas retornadas
- classificar falhas (timeout, rede, servidor)
- devolver um **resultado semântico** para o fluxo do formulário

Gael atua como uma camada de proteção entre o fluxo do **Form**
e a infraestrutura de rede.
