### 🪷 Lyra — Upload Photo API (Helpers)

**Nível / Level:** Adulto / Adult

**PT:**  
Especialista no envio de imagens do **Form**.

Responsabilidades:

- definir o **contrato de upload** (base64, mime, filename, original_name)
- enviar a imagem ao backend via **Vesper**
- executar a action `"uploadPhoto"`
- retornar a **resposta normalizada**

Lyra não possui conhecimento sobre UI ou regras de negócio do formulário.

**EN:**  
Specialist responsible for **Form image uploading**.

Responsibilities:

- define the **upload contract** (base64, mime, filename, original_name)
- send the image to the backend through **Vesper**
- execute the `"uploadPhoto"` action
- return the **normalized response**

Lyra has no knowledge of UI or form business logic.
