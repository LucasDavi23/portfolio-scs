### 🧠 Sofia — Form Validation & UX State Specialist

**Nível / Level:** Jovem / Young

**PT:**  
Especialista na **validação básica de UX do formulário** e no **estado visual durante preenchimento e envio**.

Responsabilidades:

- coletar valores dos inputs (com `trim`)
- validar **regras simples** (honeypot, campos obrigatórios, limites de texto)
- atualizar **status de feedback** (info/success/error) via `data-attribute`
- **travar e destravar o botão de envio** (`aria-busy` + alteração de texto)
- marcar e limpar **erros de campo** sem exigir mudanças no HTML

Sofia atua apenas na **validação de UX e no estado visual do formulário**.

Ela não processa imagens (**Athena**),  
não controla a UI da foto (**Daphne**),  
não controla a UI de estrelas (**Ayla**),  
não emite eventos e não chama API ou fluxo.

**EN:**  
Specialist in **basic form UX validation** and the **visual state during input and submission**.

Responsibilities:

- collect input values (using `trim`)
- validate **simple rules** (honeypot, required fields, text limits)
- update **feedback status** (info/success/error) via `data-attribute`
- **lock and unlock the submit button** (`aria-busy` + label change)
- mark and clear **field errors** without requiring HTML changes

Sofia operates only in **UX validation and form visual state**.

She does not process images (**Athena**),  
does not control the photo UI (**Daphne**),  
does not control the rating UI (**Ayla**),  
does not emit events and does not call APIs or the form flow.
