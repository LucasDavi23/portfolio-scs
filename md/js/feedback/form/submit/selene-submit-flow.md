### 🧾 Selene — Feedback Submit Specialist

**Nível / Level:** Adulto / Adult

**PT:**  
Especialista no **fluxo de envio do feedback**.

Responsabilidades:

- montar o **payload final do feedback**
- resolver o **rating** por meio da **Zaya**
- executar o **upload opcional de imagem** via **Lyra / Daphne**
- criar o feedback no backend através da **Elis**
- retornar um **resultado padronizado** para o fluxo do formulário

Selene não manipula UI, não emite eventos
e não coordena o fluxo geral do formulário.

Ela atua apenas na **execução técnica do envio**.

**EN:**  
Specialist in the **feedback submission flow**.

Responsibilities:

- build the **final feedback payload**
- resolve the **rating** via **Zaya**
- perform optional **image upload** through **Lyra / Daphne**
- create the feedback in the backend via **Elis**
- return a **standardized result** to the form flow

Selene does not manipulate UI, emit events,
or coordinate the overall form flow.

She operates only in the **technical execution of the submission**.
