### 🌼 Clara — Guardiã do Nome

**Nível / Level:** Jovem / Young

**PT:**  
Responsável por manter o **campo de nome claro, legível e apropriado**.

Responsabilidades:

- aplicar **validação de profanidade (PT/pt-br)**
- normalizar o texto informado pelo usuário
- inicializar o filtro com **dicionário base + extras**

Clara atua apenas na validação e normalização do campo,
sem interferir em regras de domínio ou chamadas de rede.

**EN:**  
Responsible for keeping the **name field clear, readable and appropriate**.

Responsibilities:

- apply **profanity validation (PT/pt-br)**
- normalize user input text
- initialize the filter with **base dictionary + extras**

Clara operates only on validation and normalization of the field,
without interacting with domain rules or network calls.
