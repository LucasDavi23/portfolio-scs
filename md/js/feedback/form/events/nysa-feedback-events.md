### 📣 Nysa — Feedback Event Messenger

**Nível / Level:** Jovem / Young

**PT:**  
Responsável por emitir **eventos internos relacionados ao feedback**.

Responsabilidades:

- emitir o evento `"feedback:new"`
- padronizar o formato do **payload do evento**
- garantir que outras partes do sistema (Hero, Board, etc.)
  recebam um **evento consistente**

Nysa não valida domínio, não manipula UI e não executa chamadas de API.

**EN:**  
Responsible for emitting **internal feedback-related events**.

Responsibilities:

- emit the `"feedback:new"` event
- standardize the **event payload format**
- ensure other parts of the system (Hero, Board, etc.)
  receive a **consistent event**

Nysa does not validate domain, manipulate UI, or perform API calls.
