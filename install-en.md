# ⚙️ Installation Guide — SCS Portfolio

## Overview

This project uses:

- Vite (build tool)
- Tailwind CSS
- Modular JavaScript
- Google Apps Script (API backend)

---

## 📌 Requirements

Before starting:

- Node.js 18+
- npm
- Modern browser

---

## 📦 Installing dependencies

`bash npm installation`

Dependencies are managed automatically by `package.json`.

---

## 🚀 Run the project (dev)

```bash.com npm execute dev

```

Access:

```
http://localhost:5173
```

---

## 🎨 Compile CSS (Tailwind)

Development mode:

```bash.compile dev:css

```

Optimized build:

```bash.compile:css

```

---

## 🏗️ Production build

```bash.compile

```

View compilation:

```bash.compile execution preview

```
